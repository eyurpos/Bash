# Fetch external data

## Curl
> sudo apt-get install curl

> curl '[url]'

> curl '[url]' -v - verbose

> curl '[url]' -o [output_file]

> curl '[url]' -L - redirect

> curl '[url]' -s - silent

> curl '[url]' > /dev/tty

## jq
> sudo apt-get install jq

> jq '.field'

> jq '[.field]'

> jq '[.field1, .field2]'
